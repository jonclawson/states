# Next.js + Zustand Sample App

This sample demonstrates common and useful patterns when using Zustand in a Next.js TypeScript app:

- Basic stores and actions (counter)
- Derived state and selectors (todos count)
- Async actions (fetch simulated data)
- Persisting state using `persist`
- Devtools integration using `devtools`
- Subscriptions and external listeners
- Server-side hydration example (SSR -> client)

Quick start

1. cd into the `zustand` folder
2. npm install
3. npm run dev

Open http://localhost:3000
