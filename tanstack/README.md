# Next.js + TanStack Query (React Query) Sample

This sample demonstrates common TanStack Query features in a Next.js TypeScript app:

- Basic queries with `useQuery`
- Mutations with `useMutation` and optimistic updates
- Pagination and `useInfiniteQuery`
- Dependent queries and prefetching
- Server-side data hydration with `dehydrate`
- Devtools integration

Quick start:
1. cd into `tanstack`
2. npm install
3. npm run dev

Open http://localhost:3000
