# Nuxt 3 + Pinia Sample

This sample demonstrates common Pinia patterns inside a Nuxt 3 app:

- Basic stores (state + getters + actions)
- Async actions and server-side hydration (SSR)
- Selective subscriptions with `$subscribe`
- Persisting store parts using `pinia-plugin-persistedstate`

Quick start

1. cd pinia
2. npm install
3. npm run dev

Open http://localhost:3000
